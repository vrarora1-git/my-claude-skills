# vibe-check

vibe-check is a free, open-source skill for Claude Code, Codex, or Antigravity. It acts as a product partner, handling foundational work before coding begins.

## Key Functions

Problem Validation: Prioritizes user pain over feature lists by vetting solutions against real-world complaints.

User Flows: Generates Mermaid diagrams for handoff to coding agents.

Tech Stack: Recommends architectures with clear rationales.

Data Modeling: Derives schemas from product flows.

Phased Build: Provides a checkpoint-based roadmap.
